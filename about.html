About the Insertion Sort Algorithm

The insertion sort algorithm takes the power value of each Mario Kart item and compares it to the value of the item before it. If the value of the current item is less than that of the previous item, it inserts itself before the previous item. The sort algorithm iterates though every item in the list putting it in order from smallest to greatest power value

What's the big O?

The big O for this insertion sort algorithm is O(n). As it is only iterating though one array of list items and altering the DOM to get them in the correct order, it can only have a maximum number of steps as long as the array.
